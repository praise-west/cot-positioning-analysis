# COT Positioning Analysis

Extreme positioning signals in Euro FX, Crude Oil, and E-mini S&P 500 futures using COT data.

## Key Findings
- Strongest long signal in Crude Oil (+2.69% 1W on dual extremes)
- Reliable contrarian short in Euro FX (-0.145% 2W on dual extremes)
